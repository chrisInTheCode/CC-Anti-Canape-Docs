CC Anti-Canape: CC DOT Parsing Application

Author: Chris Hinkson
Email: Christopher.Hinkson@clubcar.com

Desc: This application is designed to parse a Club Car DOT file and save the data as json files for easy interaction with python dictionaries. This will make it easier to use CAN communication in python, specifically writing.

dotParse

The application presents a central dotParse() class that can be used as a parsed-dot directory. It includes the following features: - Initialization of empty dictionaries to declare space usage along with empty variable lookup - parseNewDOT(filepath) method to load a given DBC into all of the dictionaries - dumpDictionaries() method to save the current dictionaries as json files in the 'dictionaries' folder - loadDictionaries() method to load the json files in the 'dictionaries' folder as current dictionaries - Several support methods for getting relevant data by lookup with hex ID, decimal ID, or parameter name

Examples:

currentDOT = dotParse() currentDOT.parseNewDOT("DiagnosticsObjectTableV14.xlsx") print(currentDOT.getParameterNameFromHexId("0x2808")) print(currentDOT.getDecIdFromHexId("0x2808")) print(currentDOT.getParameterNameFromDecId("10248")) print(currentDOT.getHexIdFromDecId("10248")) print(currentDOT.getDecIdFromParameterName("DST_rule_time_start")) print(currentDOT.getHexIdFromParameterName("DST_rule_time_start"))

class dotParse():
    ''' The application presents a central dotParse() class that can be used as a parsed-dot directory. It includes the following features:
    - Initialization of empty dictionaries to declare space usage along with empty variable lookup
    - parseNewDOT(filepath) method to load a given DBC into all of the dictionaries
    - dumpDictionaries() method to save the current dictionaries as json files in the 'dictionaries' folder
    - loadDictionaries() method to load the json files in the 'dictionaries' folder as current dictionaries
    - Several support methods for getting relevant data by lookup with hex ID, decimal ID, or parameter name

    Examples:
        currentDOT = dotParse()
        currentDOT.parseNewDOT("DiagnosticsObjectTableV14.xlsx")
        print(currentDOT.getParameterNameFromHexId("0x2808"))
        print(currentDOT.getDecIdFromHexId("0x2808"))
        print(currentDOT.getParameterNameFromDecId("10248"))
        print(currentDOT.getHexIdFromDecId("10248"))
        print(currentDOT.getDecIdFromParameterName("DST_rule_time_start"))
        print(currentDOT.getHexIdFromParameterName("DST_rule_time_start"))


    Attributes:
        dotPath: The path to the current instance's DOT file.
        dotName: The name of the current instance's DOT file.
        hexIdToParameterDict: A dictionary mapping hex ID's to parameter names.
        hexIdToDecIdDict: A dictionary mapping hex ID's to decimal ID's.
        hexIdToRangeDict: A dictionary mapping hex ID's to parameter range values.
        hexIdToOffsetDict: A dictionary mapping hex ID's to parameter offset values.
        hexIdToScaleDict: A dictionary mapping hex ID's to parameter scale values.
        hexIdToUnitDict: A dictionary mapping hex ID's to parameter units of measurement.
        hexIdToNotesDict: A dictionary mapping hex ID's to a parameter's notes.
        decIdToParameterDict: A dictionary mapping decimal ID's to parameter names.
        decIdToHexIdDict: A dictionary mapping decimal ID's to hex ID's.
        parameterToHexIdDict: A dictionary mapping parameter names to hex ID's.
        parameterToDecIdDict: A dictionary mapping paremeter names to decimal ID's.
    '''    

    ########################
    # CLASS INITIALIZATION #
    ########################
    def __init__(self) -> None:
        ''' Initializes the dotParse class upon instance creation.
        '''
        self.dotPath = None
        self.dotName = None

        # Hex ID will serve as base for all lookup attributes | Initialize Hex Dictionaries
        self.hexIdToParameterDict = defaultdict(str)
        self.hexIdToDecIdDict = defaultdict(str)
        self.hexIdToRangeDict = defaultdict(str)
        self.hexIdToOffsetDict = defaultdict(str)
        self.hexIdToScaleDict = defaultdict(str)
        self.hexIdToUnitDict = defaultdict(str)
        self.hexIdToNotesDict = defaultdict(str)

        # For lookup by Decimal ID | Initialize Decimal Dictionaries
        self.decIdToParameterDict = defaultdict(str)
        self.decIdToHexIdDict = defaultdict(str)

        # For lookup by Parameter Name | Initialize Parameter Dictionaries
        self.parameterToHexIdDict = defaultdict(str)
        self.parameterToDecIdDict = defaultdict(str)


    ##############
    # DOT LOADER #
    ##############
    def parseNewDOT(self, filePath:str):
        '''Parses a DBC file and loads all information into instance-based dictionaries.

        Args:
            filePath (str): The Path to the DBC file to parse.
        '''        
        # Save the dbc path
        self.dotPath = filePath
        # Save the dbc name
        self.dotName = path.basename(self.dotPath)

        # Load the DOT into a pandas dataframe
        dotAsDataFrame = read_excel(io=self.dotPath, sheet_name="DOT", header=0, usecols=["Object Id", "Dec", "Parameter Name", "Range", "Offset", "Scale", "Unit", "Notes"])

        # For the entire excel file
        for indexCount in dotAsDataFrame.index:
            # Sort out the "reserved" rows from excel file
            if (dotAsDataFrame['Object Id'][indexCount] == 'Reserved'):
                continue

            # Else, add all of the appropriate entries to the respective dictionaries
            self.hexIdToParameterDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Parameter Name'][indexCount]
            self.hexIdToDecIdDict[dotAsDataFrame['Object Id'][indexCount]] = str(int(dotAsDataFrame['Dec'][indexCount]))
            self.hexIdToRangeDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Range'][indexCount]
            self.hexIdToOffsetDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Offset'][indexCount]
            self.hexIdToScaleDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Scale'][indexCount]
            self.hexIdToUnitDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Unit'][indexCount]
            self.hexIdToNotesDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Notes'][indexCount]
            self.decIdToParameterDict[str(int(dotAsDataFrame['Dec'][indexCount]))] = dotAsDataFrame['Parameter Name'][indexCount]
            self.decIdToHexIdDict[str(int(dotAsDataFrame['Dec'][indexCount]))] = dotAsDataFrame['Object Id'][indexCount]
            self.parameterToHexIdDict[dotAsDataFrame['Parameter Name'][indexCount]] = dotAsDataFrame['Object Id'][indexCount]
            self.parameterToDecIdDict[dotAsDataFrame['Parameter Name'][indexCount]] = str(int(dotAsDataFrame['Dec'][indexCount]))


    #############################
    # SAVE DICTIONARIES TO JSON #
    #############################
    def dumpDictionaries(self):
        '''Saves (dumps) the current instance's dictionaries into the json save files located in the 'dictionaries' folder.

        Returns:
            bool: True if execution completed with no errors, False otherwise.
        '''     
        # [https://www.geeksforgeeks.org/create-a-directory-in-python/]
        try:
            mkdir("dictionaries")
        except OSError as error:
            print("CC DOT-AutoParse: Dictionary Directory Already Exists!")

        # Save the hexIdToParameterDict Dictionary
        with open('dictionaries\\hexIdToParameterDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToParameterDict, outputFile, indent="")

        # Save the hexIdToDecIdDict Dictionary
        with open('dictionaries\\hexIdToDecIdDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToDecIdDict, outputFile, indent="")

        # Save the hexIdToRangeDict Dictionary
        with open('dictionaries\\hexIdToRangeDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToRangeDict, outputFile, indent="")

        # Save the hexIdToOffsetDict Dictionary
        with open('dictionaries\\hexIdToOffsetDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToOffsetDict, outputFile, indent="")

        # Save the hexIdToScaleDict Dictionary
        with open('dictionaries\\hexIdToScaleDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToScaleDict, outputFile, indent="")

        # Save the hexIdToUnitDict Dictionary
        with open('dictionaries\\hexIdToUnitDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToUnitDict, outputFile, indent="")

        # Save the hexIdToNotesDict Dictionary
        with open('dictionaries\\hexIdToNotesDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.hexIdToNotesDict, outputFile, indent="")

        # Save the decIdToParameterDict Dictionary
        with open('dictionaries\\decIdToParameterDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.decIdToParameterDict, outputFile, indent="")

        # Save the decIdToHexIdDict Dictionary
        with open('dictionaries\\decIdToHexIdDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.decIdToHexIdDict, outputFile, indent="")

        # Save the parameterToHexIdDict Dictionary
        with open('dictionaries\\parameterToHexIdDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.parameterToHexIdDict, outputFile, indent="")

        # Save the parameterToDecIdDict Dictionary
        with open('dictionaries\\parameterToDecIdDict.json', 'w') as outputFile:
            # [https://docs.python.org/3/library/json.html#json.dump]
            dump(self.parameterToDecIdDict, outputFile, indent="")


    ###############################
    # LOAD DICTIONARIES FROM JSON #
    ###############################
    def loadDictionaries(self):
        ''' Loads the json save files located in the 'dictionaries' folder into the instance's dictionaries.

        Returns:
            bool: True if execution completed with no errors, False otherwise.
        '''   
        # Load the hexIdToParameterDict Dictionary
        with open('dictionaries\\hexIdToParameterDict.json') as hexIdToParameterDict:
            # [https://docs.python.org/3/library/json.html#json.load]
            self.hexIdToParameterDict = load(hexIdToParameterDict)

        # Load the hexIdToDecIdDict Dictionary
        with open('dictionaries\\hexIdToDecIdDict.json') as hexIdToDecIdDict:
            # [https://docs.python.org/3/library/json.html#json.load]

        # Load the hexIdToOffsetDict Dictionary

def parseNewDOT(self, filePath:str):
    '''Parses a DBC file and loads all information into instance-based dictionaries.

    Args:
        filePath (str): The Path to the DBC file to parse.
    '''        
    # Save the dbc path
    self.dotPath = filePath
    # Save the dbc name
    self.dotName = path.basename(self.dotPath)

    # Load the DOT into a pandas dataframe
    dotAsDataFrame = read_excel(io=self.dotPath, sheet_name="DOT", header=0, usecols=["Object Id", "Dec", "Parameter Name", "Range", "Offset", "Scale", "Unit", "Notes"])

    # For the entire excel file
    for indexCount in dotAsDataFrame.index:
        # Sort out the "reserved" rows from excel file
        if (dotAsDataFrame['Object Id'][indexCount] == 'Reserved'):
            continue

        # Else, add all of the appropriate entries to the respective dictionaries
        self.hexIdToParameterDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Parameter Name'][indexCount]
        self.hexIdToDecIdDict[dotAsDataFrame['Object Id'][indexCount]] = str(int(dotAsDataFrame['Dec'][indexCount]))
        self.hexIdToRangeDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Range'][indexCount]
        self.hexIdToOffsetDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Offset'][indexCount]
        self.hexIdToScaleDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Scale'][indexCount]
        self.hexIdToUnitDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Unit'][indexCount]
        self.hexIdToNotesDict[dotAsDataFrame['Object Id'][indexCount]] = dotAsDataFrame['Notes'][indexCount]
        self.decIdToParameterDict[str(int(dotAsDataFrame['Dec'][indexCount]))] = dotAsDataFrame['Parameter Name'][indexCount]
        self.decIdToHexIdDict[str(int(dotAsDataFrame['Dec'][indexCount]))] = dotAsDataFrame['Object Id'][indexCount]
